# Animal Sudoku

A fun web Sudoku game using animal icons, with drag/drop and pencil notes.

## Features
- Drag & drop animal placement
- Click placement
- Pencil mode
- Undo & Hint
- Keyboard support
- Sound toggle
- Dark & light themes
- Persistent settings
- Share puzzles via URL

## Run Locally
```bash
npm install
npm run dev
